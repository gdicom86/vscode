변수나 상수에 할당할 수 있는 값의 종류

  • 숫자: Number => 정수 실수 NaN(Not a Number: 숫자가 아니라는 값)
  • 문자: String => "", '', ``으로 감싼 값들
  • 불리언: Boolean => true / false
  • 객체: object => 1가 이상의 데이터가 특정한 구조를 가지고 있는 형태
  • 배열: Array => [1, 2, 3, 4, ...] 형태로 이루어진 데이터 집합(순서, 길이 있음)
  • 알수 없는 값: null => 있긴한데 파악이 안됨
  • 미정된 값: undifined: 값이 할당되지 않음
  • 데이터 타입 확인 명령어: typeof

변수의 관리

  1. 기본 자료형(primative type): 값이 메모리의 주소 값이 가르키는 곳에 직접할당
    number, boolean, string(원래는 참조형 자료형이지만 사용편의성을 위해 같이 사용가능)
  2. 참조 자료형(reference type): 값이 지정되어있는 메모리 주소값을 참조하고 잇는 할당
    string, object, function, null, undefined
