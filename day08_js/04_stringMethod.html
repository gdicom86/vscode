가장 많이 다루는 데이터 타입: string

  • 문자열 데이터는 index(0부터시작)를 가지고 있음.
  • 문자열 데이터 값 내에서는 개행(br)하는것을 추천하지 않는다.
  • new String(); 객체 타입으로 선언은 가능하지만 추천하지 않음
  • length: 문자열의 총길이 => index는 length-1

문자관련 메서드: java에서 사용하는 String 메서드와 비슷

  • 문자 추출 관련 메서드: substring(index, index), substr(), slice()
  • 문자 대체 메서드: replace(old, new): 첫번째 매칭되는 문자만 replace, 대소문자 구분
  • 문자열 양측 공백제거: trim()
  • padStart(목표문자길이, 채울문자) / padEnd: 앞쪽 / 뒤쪽에서 채우기
    문자값의 힌트, 010101 - *****
  • 해당 index 문자 추출: charAt(index)
  • 문자열의 위치를 반환: indexOf('str');
  • startWith('str') / endWith('str'): 지정 문자열이 탐색대상 문자열에서 시작/끝에 존재하는지 확인하는 메서드
    return true / false
